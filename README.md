# Personal Task Tracker (Java)

A simple console-based application built with Java to manage daily tasks. 
It uses File I/O to save tasks permanently in a `.txt` file.

## Features
- Add new tasks.
- View all saved tasks.
- Persistent storage using local file system.

## How to Run
1. Download `TaskTracker.java`.
2. Open terminal and run: `javac TaskTracker.java`
3. Run the app: `java TaskTracker`
